# Notes
# ============================================

# Local Development Setup:
# 1. Copy this file: cp env.example.txt .env.local
# 2. Update DATABASE_URL with your local PostgreSQL connection
# 3. Keep Supabase credentials for authentication (can be same as production)

# Production Setup:
# 1. Copy this file: cp env.example.txt .env.production (don't commit!)
# 2. Update DATABASE_URL with Supabase production connection (Transaction Pooler)
# 3. Update DIRECT_URL with Supabase direct connection (for migrations)
# 4. Update all Supabase keys from your project dashboard
# 5. Set NODE_ENV="production"

# Getting Supabase Connection Strings:
# 1. Go to: https://app.supabase.com/project/[PROJECT-REF]/settings/database
# 2. Find "Connection string" section
# 3. Use "Transaction" mode for DATABASE_URL (pgbouncer=true)
# 4. Use "Direct connection" for DIRECT_URL
# 5. Replace [YOUR-PASSWORD] with your actual database password
